No CSS nós possuímos os seletores por ID

  • Selecionador por Tipo:Busca elementos por uma tag específica
  • seletor por ID(a):Busca elementos através do atributo *id*
  • Seletor por classe(.):Busca elementos através do atributo Busca elementos atraves do atributo *class*
  • Seletor Universal(*):seleciona todos os elementos HTML
  • Seletor de atributo ([atrib]):Secionas elementos que possuem um atributo especifico em nosso documento HTML e consiguims tambem buscar atributos com um valor especifico.
